# 英語

私ぐらいの歳の人って、英語はだいたいの人が12歳ぐらいで習い始めたと思うんですが、わたしいま55なんでもう40年ですね。英語の学習って、中高6年間と、大学で数単位習った以外は自習しかしてないんですが、それでTOEIC915点なのでまあまあえらいな、と我ながら思うわけですけど ^^;

いま40年間でかつてなく英語学習のモチベーションが上がってるんですよ。問題点は3つ。

- 語彙が足りない。コンピュータ系の語彙とか、論文の語彙は豊富なんだけど、普通の語彙 (ritualとかpodiumとか) が足りない。これはふつうに英語を読むしかないですね。
- 喋るのはまあまあ喋るんですけど、アクセントが日本語風なのももちろんあるし、それ以上に英語の慣用がわからないので、火星から来た人みたいな独特の英語になってしまう。まあこれはしかたないかな...
- 話された英語が聞こえない。タイ語は聞き習いをしたので逆に文字は読めないけど音はわかるわけですけど、英語は圧倒的に「聞く」体験が少ないので、聞こえた音がなんなのか、わからん。これは聞くしかない。

というわけで、「読む」「聞く」をたくさんやる、という、普通の結論に達しました ^^;
目標は、こういうのが「ふつうに」きこえるようになることです ^^;
