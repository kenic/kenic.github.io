# 航空会社ラウンジ

20年ぐらい前にステータスとって(会社持ちの出張とかじゃなくて、自腹で)、それからずっとスーパーフライヤーズなんですけど、28年から改悪? されて、SFCヒラ会員はカードで300万円以上決済しないとスタアラシルバーになっちゃうらしい。300万円も行けるかなあ... 利用額のカウントは26年の12月からだそうです。もうすぐ!

まあ、20年もラウンジ使わせてもらったから、たいへんありがたかったな、という気はします。
