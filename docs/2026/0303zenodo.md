# Zenodo

というわけで、arXiv はあきらめて、論文は[Zenodo](https://zenodo.org)に置くことにしました。arXivは「プレプリントだけ」「毎日巡回して掲載された論文をチェックする」みたいにして使うもので、Zenodoは「いろんなもの」(図、データセット、プログラム、プレプリント、ジャーナルペーパー)を置いておくようの物で用途が違う。まあどこかに置いておければいいので、Zenodoでも、ResearchGateでも構わないです。ZenodoはDOIがつくのでリンクするのに便利かもしれないです。

というわけで下記に置いておきました。パチパチ!
